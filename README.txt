ScaffR200 Pro v5.7

Nouveautés
- Un clic sur une vue ouvre le plan en plein écran.
- Gestes tactiles en plein écran :
  * pincer avec deux doigts pour zoomer ;
  * déplacer les vues 2D avec un doigt ;
  * faire pivoter et incliner la vue 3D avec un doigt ;
  * zoom à la molette sur ordinateur.
- Diagonales dessinées à chaque niveau, avec alternance du sens.
- Plinthes représentées à chaque niveau de travail.
- Les échafaudages dépassent d'une largeur d'échafaudage à chaque extrémité.
- Des retours de plancher sont dessinés aux angles afin de représenter la
  circulation continue autour des quatre façades.
- La vue 3D complète de la V5.6 est conservée.

Installation sur GitHub
1. Décompresser le ZIP.
2. Dans GitHub : Add file, puis Upload files.
3. Glisser index.html, README.txt et le dossier assets.
4. Cliquer sur Commit changes.
5. Attendre la mise à jour de GitHub Pages.

Avertissement
Ces vues sont des représentations de préparation. Elles ne remplacent pas
une étude de stabilité, une note de calcul ou un plan validé par le fabricant.
