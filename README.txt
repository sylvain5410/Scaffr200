ScaffR200 Pro v5.1

Évolutions :
- largeur 0,73 m proposée par défaut pour chaque façade ;
- sélection de plusieurs éléments de départ simultanément ;
- exemple possible : manchons de départ + montants de 0,50 m ;
- option « Rehausse au dernier étage » ;
- rehausse : 1 montant de 1 m par ligne de cadre ;
- rehausse : 2 lisses supplémentaires par travée ;
- conservation des cadres asymétriques au dernier niveau ;
- cales bois en quantité égale aux socles à vis.

Règles de quantité pour le départ :
- cadre 2 m ou cadre de départ : 1 pièce par ligne de cadre ;
- montant ou manchon de départ : 2 pièces par ligne de cadre ;
- plusieurs sélections s'additionnent.

Installation GitHub Pages :
1. Décompresser le fichier ZIP.
2. Remplacer l'ancien index.html par celui de la V5.1.
3. Valider avec « Commit changes ».
4. Attendre la mise à jour de GitHub Pages.

Attention :
Les références marquées AUTO restent provisoires. Le calcul doit toujours être
contrôlé selon la notice fabricant et les particularités du chantier.
