ScaffR200 Pro V6.0

Première version de la nouvelle architecture professionnelle.

Nouveautés
- HTML, CSS, moteur géométrique et application séparés.
- Modèle unifié : chaque montant, plancher, lisse, plinthe et diagonale devient un objet géométrique.
- Toutes les façades sont reconstruites à chaque modification depuis la même source de données.
- Compteur d’objets du moteur V6 visible sous les plans.
- Conservation de toutes les fonctions V5.10 : façades, matériel, vues 2D/3D, gestes tactiles, dépassements et rehausses.
- Compatibilité avec les sauvegardes V5.10.

Installation
- Version complète : publier tous les fichiers en conservant leur arborescence.
- Version simple : remplacer uniquement index.html.

Note
Cette V6.0 pose le nouveau moteur objet. Les prochaines versions pourront migrer progressivement chaque moteur de rendu et la nomenclature vers ces objets sans casser l’interface.
