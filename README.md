# ScaffR200 V1.0

Application web installable sur iPhone pour préparer des chantiers en R200 Progress acier.

## Fonctions

- Longueur personnalisable pour chaque travée
- Ajout, suppression et duplication de travées
- Types de travée : standard, accès, angle
- Calcul automatique de la longueur, hauteur et surface
- Plan 2D
- Vue 3D simplifiée avec rotation au doigt
- Estimation du matériel regroupée par longueur
- Sauvegarde locale
- Import et export des chantiers en JSON
- Impression / export PDF
- Fonctionnement hors connexion après la première ouverture

## Mise à jour GitHub

1. Décompresser `ScaffR200_V1.0.zip`.
2. Dans le dépôt GitHub, cliquer sur **Add file > Upload files**.
3. Envoyer tous les fichiers contenus dans le dossier.
4. Accepter le remplacement des anciens fichiers.
5. Cliquer sur **Commit changes**.
6. Attendre quelques minutes puis rouvrir l’application sur l’iPhone.

## Important

Les quantités sont des estimations de préparation. Elles ne remplacent pas la notice fabricant, une étude de chantier, une note de calcul ou la validation par une personne compétente.
